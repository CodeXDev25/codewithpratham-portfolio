# Mangnale Prathamesh — Portfolio
 
> Personal portfolio website built with React & Tailwind CSS, following a Minimalist Monochrome editorial design system.
 
## Overview
 
A performance-first, fully responsive personal portfolio for **Mangnale Prathamesh** — First Year B.Tech Computer Engineering student at Vishwakarma Institute of Technology, Pune.
 
Designed around a strict black-and-white editorial aesthetic inspired by high-end fashion publications and architectural monographs. No accent colors. No gradients. Just typography, contrast, and negative space.

# Tech Stack
 
| Layer | Technology |
|---|---|
| Framework | React 18 |
| Styling | Tailwind CSS + inline CSS variables |
| Fonts | Google Fonts (Playfair Display, Source Serif 4, JetBrains Mono) |
| Deployment | Cloudflare Pages |
| Version Control | Git + GitHub |
